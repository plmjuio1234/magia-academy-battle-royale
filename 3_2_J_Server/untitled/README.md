# 유급은 싫어 - 게임 서버

## 🚀 서버 실행 방법 (Windows)

### 1단계: 서버 컴파일

```bash
cd 3_2_J_Server/untitled
javac -encoding UTF-8 -cp "lib/*" src/main/java/org/example/*.java -d out
```

### 2단계: 서버 실행

```bash
java -cp "out;lib/*" org.example.Main
```

### ⚠️ 주의사항

- **반드시 `out/` 폴더를 사용하세요** (Maven의 `target/` 폴더는 삭제됨)
- 코드 수정 후에는 **1단계부터 다시 실행**하여 재컴파일 필요
- IntelliJ IDEA에서 실행하지 마세요 (빌드 충돌 가능성)

## 📋 서버 정보

- **포트**: TCP 5000, UDP 5001
- **게임 틱**: 20Hz (50ms)
- **최대 플레이어**: 8명

## ⚙️ 서버 기능

- ✅ 방 생성/참가/나가기
- ✅ 방 목록 조회
- ✅ 게임 시작
- ✅ 플레이어 위치 동기화
- ✅ 몬스터 스폰 및 AI
- ✅ 스킬 시전
- ✅ 채팅

## 🔧 필수 요구사항

- Java 11 이상
- Maven (선택)

## 📝 서버 실행 확인

서버가 정상적으로 실행되면 다음 메시지가 출력됩니다:

```
=================================
  방 기반 게임 서버 시작!
  TCP 포트: 5000
  UDP 포트: 5001 (미사용)
=================================
```
