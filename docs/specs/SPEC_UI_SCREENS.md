## 🎮 화면 구조

### LoadingScreen
- **목적**: 게임 리소스 로드
- **주요 요소**:
  - 로고 중앙 배치
  - 로딩 프로그레스 바 (0~100%)
  - 팁 텍스트 (무작위)
  - 상태 메시지 ("폰트 로딩 25%", "맵 로딩 60%" 등)

### MainMenuScreen
- **목적**: 게임 시작 메뉴
- **주요 버튼**:
  - [시작]: LobbyScreen으로
  - [설정]: 음량, 그래픽 설정
  - [종료]: 게임 종료

### LobbyScreen
- **좌측 (2/3)**: 방 목록
  - 방 제목, 인원수, 참가 버튼
  - 스크롤 가능
  - [새로고침], [타이틀로돌아가] 버튼

- **우측 (1/3)**: 캐릭터 커스터마이징
  - 캐릭터 미리보기 (중앙)
  - 닉네임 입력칸
  - [외형변경] 버튼
  - [게임시작] 버튼 (방 참가 후 활성화)

### WaitingRoomScreen
- **상단 (1/2)**: 플레이어 정보
  - 최대 4명의 캐릭터 미리보기 + 닉네임
  - 2x2 그리드 배치

- **하단 (1/2)**:
  - **좌측**: 채팅창 (스크롤 가능), 입력창
  - **우측**: [게임시작] (호스트만), [방나가기]

### GameScreen
- **좌측 상단**: 플레이어 정보
  - Lv: 5
  - HP: ████░ (70/100)
  - MP: ██░░░ (40/50)
  - 배경: 반투명 검정

- **중앙**: 게임 월드
  - 플레이어 (화면 중앙 고정)
  - 다른 플레이어들
  - 몬스터들
  - 스킬 이펙트들
  - 맵 경계

- **우측 상단**: 킬 로그
  - 최근 5개 킬
  - 자동 스크롤
  - 반투명 배경

- **좌측 하단**: 조이스틱
  - 크기: 화면 너비의 18~20%
  - 투명도: 50%
  - 드래그로 조작

- **우측 하단**: 스킬 버튼
  - 4개 버튼 (2x2 그리드)
  - A, B, C: 현재 스킬
  - OFF: 자동 공격 토글
  - 마나 부족 시 회색 처리

### ResultScreen
- **배경**: 게임 화면 어둡게 처리
- **중앙 모달**:
  - 순위 표시: "1등 - 졸업 성공! 🎓"
  - 메시지: 순위별 다른 메시지
  - 통계 (선택사항):
    - 플레이 시간
    - 몬스터 처치 수
    - 플레이어 처치 수
  - [타이틀로돌아가기] 버튼

## 🎨 색상 & 스타일

### 색상 팔레트
```
배경: #1a1a1a (어두운 회색)
텍스트: #ffffff (흰색)
강조: #ff6b6b (빨강) / #4ecdc4 (청록)
UI 배경: rgba(0, 0, 0, 0.7) (반투명)

원소별:
  불: #ff4444 (빨강)
  물: #4488ff (파랑)
  바람: #44ff88 (초록)
  땅: #884444 (갈색)
  전기: #ffff44 (노랑)
```

### 폰트
- **기본**: NotoSansCJK (한글 지원)
- **타이틀**: 판타지 테마 폰트 (선택사항)

## 📐 레이아웃 가이드

### 반응형 설계
- **기준 해상도**: 1080x1920 (9:16)
- **모든 요소**: 백분율 기반 위치 지정
- **조이스틱 크기**: 너비의 18~20%
- **버튼 간격**: 터치 오류 방지 (최소 8dp)

### 안전 영역 (Safe Area)
- 상단: 24dp (상태바 고려)
- 하단: 24dp (내비게이션바 고려)
