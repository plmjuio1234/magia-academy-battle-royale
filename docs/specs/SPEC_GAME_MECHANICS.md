## ⏱️ 게임 진행

**게임 시간**: 10분 (600초)
**게임 루프**: 60fps (1/60초마다 update)
**서버 주기**: 20Hz (50ms마다)

---

## 📊 경험치 & 레벨

### 레벨 시스템
```
- 최대 레벨: 15
- 시작 레벨: 1
- 시작 경험치: 0

레벨별 필요 경험치 (예시):
  Lv1→2: 50 exp
  Lv2→3: 75 exp
  ...
  Lv14→15: 500 exp

공식: 필요경험치 = 50 * (1.3^(레벨-1))
```

### 몬스터 경험치
- **고스트**: 10 exp
- **슬라임**: 25 exp
- **골렘**: 50 exp

### 레벨업 보상
```
HP +8
MP +10
ATK +10% (공격력의 10%)
DEF +2
SPD +5% (이동속도의 5%)
```

---

## ❤️ 체력/마나 시스템

### 초기값
```
HP: 100 (최대값: 100 + 8 * (레벨 - 1))
MP: 50 (최대값: 50 + 10 * (레벨 - 1))
```

### 자동 회복
- **회복 주기**: 10초마다
- **체력 회복**: 최대 체력의 3%
- **마나 회복**: 최대 마나의 5%

### 피해 시 회복 (옵션)
- 몬스터 공격 시 일정 비율 흡수 (미정)

---

## 🎯 능력치 체계

| 능력치 | 초기값 | 레벨당 증가 | 설명 |
|--------|--------|-----------|------|
| HP | 100 | +8 | 체력 |
| MP | 50 | +10 | 마나 |
| ATK | 15 | +10% | 공격력 |
| DEF | 3 | +2 | 방어력 |
| SPD | 기준값 | +5% | 이동속도 |

### 방어력 계산
```
감소_데미지 = 방어력 * 2
최종_데미지 = max(1, 받은_데미지 - 감소_데미지)
```

---

## 💥 데미지 계산

### 공격 공식
```
기본_데미지 = 공격자_ATK + 스킬_기본_데미지
최종_데미지 = 기본_데미지 - (대상_DEF * 2)
최종_데미지 = max(1, 최종_데미지)  // 최소 1 데미지
```

### 타입 보정 (추후 구현)
- 불 > 얼음
- 물 > 불
- 바람 > 물
- 땅 > 전기
- 전기 > 바람

## 🌍 맵 구조

**맵 크기**: 1920 x 1920 픽셀
**구역 수**: 4개 (초기) → 최대 5개
**구역 형태**: 사각형 분할

---

## 🔥 자기장 시스템

**시간 경과에 따른 폐쇄**:
```
0:00~2:00   모든 구역 개방
2:00~4:00   1구역 폐쇄
4:00~6:00   2구역 폐쇄
6:00~8:00   3구역 폐쇄
8:00~10:00  1개 구역만 남음
```

**폐쇄 구역 페널티**:
- 초당 10 HP 지속 손상
- 구역을 벗어나면 손상 정지

---

## 👾 몬스터 스폰

### 시간대별 스폰
```
0:00~2:00   고스트 위주
2:00~5:00   고스트 + 슬라임
5:00~8:00   슬라임 위주 + 골렘 간헐적
8:00~10:00  골렘 증가
```

### 스폰 규칙
- **위치**: 맵 중앙(960, 960)에서 300픽셀 이상 거리
- **빈도**: 1초마다 1마리 스폰
- **최대**: 방당 50마리 유지
