```
│     (Player)        ○ (Other Player)                 │

│                     ~ (Hazard Zone)                 │

│                                                      │
│                                                      │

│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐                  │
│  │ 스킬1 │ │ 스킬2 │ │ 스킬3 │ │ 스킬4 │  Lvl: 3    │
│  │⏱️0.2 │ │⏱️1.5 │ │⏱️0.0 │ │⏱️2.3 │  EXP: 45%  │
│  └─────┘ └─────┘ └─────┘ └─────┘                  │
│                                                      │
└──────────────────────────────────────────────────────┘
```

### HUD 구성

#### 상단 (고정)
- **타이머**: 게임 경과 시간 (MM:SS)
- **HP 바**: 현재 HP / 최대 HP + 수치
- **MP 바**: 현재 MP / 최대 MP + 수치

#### 하단 (고정)
- **스킬 버튼**: 4개 (크기: 100×100 px)
  - 배경색: 원소별 색상
  - 쿨다운 표시: 원형 프로그레스 바
  - 글자: 스킬 이름 + 남은 시간

#### 우측 (고정)
- **미니맵**: 축소된 맵 + 플레이어 위치
- **레벨/경험치**: 현재 레벨 + 경험치 바

#### 중앙 (동적)
- **데미지 텍스트**: 빨강색, 위로 떠오르며 사라짐
- **레벨업 알림**: "레벨업! Lv.4"
- **아이템 획득 알림**: "+30 HP"

## 입력 처리

### 터치 입력 (Android)

| 입력 | 동작 |
|------|------|
| 조이스틱 드래그 | 플레이어 이동 |
| 스킬 버튼 터치 | 스킬 사용 |
| 화면 터치 (조이스틱 외) | 드래그 방향으로 스킬 시전 |
| 2초 길게 누르기 | 인벤토리 열기 |

### 데스크톱 입력 (Keyboard + Mouse)

| 입력 | 동작 |
|------|------|
| WASD / 화살표 | 플레이어 이동 |
| 숫자 1-4 | 스킬 사용 (1번-4번) |
| 마우스 이동 | 마우스 위치로 조준 |
| 마우스 클릭 | 클릭 위치로 스킬 사용 |
| I | 인벤토리 토글 |

## 애니메이션 및 피드백

### 전환 효과
- 화면 전환: 페이드 인/아웃 (0.3초)
- 버튼 클릭: 축소 애니메이션

### 캐릭터 피드백
- 이동: 스무드 이동 (보간)
- 스킬 사용: 0.2초 딜레이 후 발동
- 피해 입음: 캐릭터 깜빡임 (0.2초)

### UI 피드백
- 스킬 쿨다운: 원형 프로그레스 바
- HP 감소: 바 애니메이션 + 붉은 번쩍임
- 아이템 획득: 팝업 텍스트 + 사운드
