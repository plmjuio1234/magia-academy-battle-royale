# 게임 컨셉 - 마법학교 배틀로얄

## 플랫폼 및 배포

- **주 타겟**: Android (libGDX Android 모듈)
- **개발/테스트**: Desktop (libGDX Desktop 모듈)
- **네트워킹**: 클라이언트-서버 모델 (Kryonet)
- **동시 플레이어**: 4명 (Kryonet 안정성 고려)

## 게임 세션 구조

- **매칭 시간**: 최대 60초 대기
- **게임 시간**: 10분
- **맵 크기**: 1920x1920 픽셀
- **틱 레이트**: Kryonet 권장사항 적용

## 게임 목표

- 최후의 1명까지 생존하기
- 동적 맵 축소 시스템 생존하기
- 몬스터 처치 및 아이템 획득으로 강화하기

## 핵심 메커니즘

1. **플레이어 이동**: 가상 조이스틱으로 자유로운 이동
2. **스킬 사용**: 원소별 5가지 스킬 시스템
3. **몬스터 퇴치**: 3종류 몬스터 처치로 아이템 획득
4. **맵 축소**: 시간에 따른 안전지역 축소 → 강제 전투 유도
5. **레벨업**: 경험치 획득으로 플레이어 강화

## 게임 페이즈

| 페이즈 | 시간 | 이벤트 | 피해 |
|--------|------|--------|------|
| Phase 0 | 0-3분 | 전체 맵 사용 가능 | 없음 |
| Phase 1 | 3-6분 | 외곽 구역 독가스 | 초당 5 |
| Phase 2 | 6-8분 | 중간 구역까지 축소 | 초당 10 |
| Phase 3 | 8-10분 | 중앙 홀만 남김 | 초당 20 |
